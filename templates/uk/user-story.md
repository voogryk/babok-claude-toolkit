### Чекліст INVEST
- [ ] **Independent** (Незалежна) — Можна розробляти без інших історій
- [ ] **Negotiable** (Обговорювана) — Деталі відкриті для обговорення з командою
- [ ] **Valuable** (Цінна) — Приносить цінність користувачу або бізнесу
- [ ] **Estimable** (Оцінювана) — Команда має достатньо інформації для оцінки
- [ ] **Small** (Мала) — Вміщується в один спринт
- [ ] **Testable** (Тестована) — Критерії приймання піддаються перевірці
